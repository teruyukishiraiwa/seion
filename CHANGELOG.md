# 変更履歴 / Changelog

本プロジェクトの主要な変更を記録します。バージョンは [Semantic Versioning](https://semver.org/lang/ja/) に従います。

## [1.1.0] - 2026-07-03

### 追加 (Added)

- **文字設定のSNS画像への反映**: 文字サイズ・行間・文字間の設定を、投稿比率表示・SNSプレビュー・PNG出力・ページ分割のすべてで共有するようにしました。画面と出力が常に同じ組み方になります（WYSIWYG）。本文幅はこれまでどおりエディタのみに適用されます。
- **タイトル表示の切り替え**: カード・出力画像へのタイトル掲載をON/OFFできる設定を追加しました（インスペクタの「SNS用画像出力」）。非表示時はタイトル分の余白が本文に開放され、ページ分割にも反映されます。エディタのタイトル入力欄は維持されます。
- **OGP / Twitter Card メタデータ**: リンク共有時にプレビュー画像付きカードが表示されるよう、Open Graph と Twitter Card のメタタグ、および OGP 画像を追加しました。

### 修正 (Fixed)

- 投稿比率表示で新規の無題ノートを作成した際、プレースホルダーがクリックを遮り本文を入力できなかった不具合を修正しました。
- ダークテーマ＋白レイヤーのカードで、テキスト選択範囲が背景に溶けて見えづらかった問題を修正しました。選択色はテーマではなくレイヤーモードに追従するようになりました。
